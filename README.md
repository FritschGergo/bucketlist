# 🎯 Bucketlist

> *Your personal dream tracker and life goals companion*

A beautiful Flutter mobile application designed to help you capture, organize, and track all the adventures and goals on your bucket list. Transform your dreams into actionable items and celebrate each accomplishment!

---

## ✨ Features

- 📝 **Create & Manage Goals** - Add new bucket list items with descriptions and details
- ✅ **Track Progress** - Mark items as completed and celebrate your achievements
- 🎨 **Beautiful UI** - Modern, intuitive design for seamless user experience
- 📱 **Mobile First** - Optimized for iOS and Android platforms
- 💾 **Persistent Storage** - Your goals are saved locally on your device
- 🔍 **Easy Navigation** - Quickly find and manage your bucket list items

---

## 🚀 Getting Started

### Prerequisites

- Flutter SDK ([Install Flutter](https://flutter.dev/docs/get-started/install))
- Dart SDK (comes with Flutter)
- An IDE (VS Code, Android Studio, or IntelliJ)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/FritschGergo/bucketlist.git
   cd bucketlist
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

### Platform-Specific Setup

**iOS:**
```bash
cd ios
pod install
cd ..
flutter run
```

**Android:**
- Ensure you have Android SDK installed
- Create a virtual device or connect a physical device
- Run `flutter run`
